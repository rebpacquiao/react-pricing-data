# React Pricing Data

### Prerequisites

- Node.js
- npm or yarn
- Tailwind
- Vite
- Firebase

### Installation

1. Clone the repository:
   git clone https://github.com/rebpacquiao/react-pricing-data.git
   cd your-project
   npm install
   npm run dev
   npm run build
